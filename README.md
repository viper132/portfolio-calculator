**Server:** Github Pages


## Run Locally

Clone the project

```bash
  git clone https://github.com/viper132/portfolio-calculator.git
```

Go to the project directory

```bash
  cd portfolio-calculator
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
